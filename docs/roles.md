# Roles

## Current Build
**No authentication in this build.** Everyone is a public visitor.

## Future Roles (For Reference)

### Crew Member (Renter)
- Browse available vehicles
- Book vehicles
- Rate and review
- Earn referral bonuses
- Access booking history

### Fleet Owner (Lessor)
- Create and manage vehicle listings
- Set pricing and availability
- Accept/reject bookings
- Upload vehicle photos and documents
- View earning history
- Manage vehicle inventory

### Admin
- Manage user verification (approve crew members)
- Moderate referral program
- View platform analytics
- Resolve disputes
- Manage site content (FAQ, testimonials, contact channels)

## Verification Requirements (Crew Members)
- Valid airline employee email or badge verification
- Background check (TBD)
- Age requirement (typically 21+)
- Valid driver's license

## Verification Requirements (Fleet Owners)
- All crew member requirements, plus:
- Vehicle registration and insurance proof
- Additional background check
- Minimum rating or referral history
